

This repo is based on a template from the below source: 

(https://www.devportfoliotemplates.com/portfolio-templates/simple-dark)


For development:
- requires Node version > 21.0.0
- 'yarn install' dependendencies
- 'yarn dev' to develop locally


To update gh-pages:
- 'yarn build'
- 'yarn predeploy'
- 'yarn deploy'
